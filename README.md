# 🚗 Driver Drowsiness Detection & Monitoring System

An AI-powered Driver Monitoring System built using OpenCV, MediaPipe, and Computer Vision techniques to enhance road safety through real-time driver alertness monitoring.

## 🎯 Project Overview

Driver fatigue and distraction are among the leading causes of road accidents. This project continuously monitors the driver's facial features and behavioral patterns to identify signs of drowsiness, microsleep, yawning, and distraction, providing timely alerts to improve driving safety.

## ✨ Key Features

* 👁️ Real-Time Eye Blink Monitoring
* 😴 Drowsiness Detection
* 🚨 Microsleep Detection with Alarm System
* 🥱 Yawning Detection
* 👀 Gaze Tracking (Left / Right / Forward)
* 📱 Phone Distraction Detection
* 🔽 Head Down Detection
* 📊 PERCLOS (Percentage of Eye Closure) Monitoring
* 🔔 Real-Time Audio Alerts

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe Face Landmarker
* NumPy
* Computer Vision
* Facial Landmark Detection

## 📈 Working Principle

The system analyzes facial landmarks in real time using MediaPipe. Eye Aspect Ratio (EAR), Mouth Aspect Ratio (MAR), gaze direction, and head posture are continuously monitored to determine the driver's alertness level and trigger warnings when unsafe behavior is detected.
